## Installation Instructions
1. Download the zip or clone the repo via above link.  
2. Choose install method: use the installer script, Docker, or build from source.  
3. For script: run `curl … prysm.sh && chmod +x prysm.sh`, then `./prysm.sh beacon-chain …` to start beacon node.  
4. For Docker: pull official Prysm images and run beacon and validator in containers.  
5. Set up JWT authentication and sync using checkpoint for fast, secure onboarding.
